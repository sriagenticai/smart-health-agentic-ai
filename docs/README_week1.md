Scope and Boundaries 

Allowed: ingest telemetry, anomaly detection, risk scoring, create clinician tasks, schedule follow-ups, notify caregivers. 

Stop/ask human: medication changes, irreversible actions, high-risk escalations. 

Out of scope: prescribing, replacing emergency services protocols. 

User Types 
Patient, Caregiver, Nurse/Physician, Care coordinator. 

Formal Conversation Samples 

Agent: “Weight gain trend suggests fluid retention—schedule tele-visit?” → clinician approves → booked. 

Agent: “Critical threshold crossed—initiate emergency escalation?” → clinician confirms → action logged.
